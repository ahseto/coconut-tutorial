Write some code that will count up the number of aluminum, paper, and plastic items in this list:

items = iter(["aluminum", "plastic", "plastic", "plastic", "plastic", "paper", "plastic", "aluminum", "aluminum", "paper", "plastic"])

Show Solution:

items = iter(["aluminum", "plastic", "plastic", "plastic", "plastic", "paper", "plastic", "aluminum", "aluminum", "paper", "plastic"]) numAluminum = 0 numPaper = 0 numPlastic = 0 def countItems(recyclable): """count the items based on material""" case (recyclable): match recyclable == "aluminum": numAluminum += 1 match recyclable is paper: numPaper += 1 match recyclable is plastic: numPlastic += 1 else: raise TypeError("item not recyclable") while (items.hasNext()) : countItems(items.next()) "Number of aluminum items: "+numAluminum.toString() |> print "Number of paper items: "+numPaper.toString() |> print "Number of plastic items: "+numPlastic.toString() |> print
